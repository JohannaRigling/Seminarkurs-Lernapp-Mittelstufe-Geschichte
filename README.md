# 🏰 HistoLearn - Geschichte Lernapp

Eine gamifizierte Lernplattform für Geschichte (Klasse 8-10) mit KI-Unterstützung, adaptivem Lernsystem und 320+ Übungen.

---

## 🚀 App starten

### Lokale Version öffnen:

```bash
# Im Browser öffnen:
file:///home/mvlg5678/Seminarkurs-Lernapp-Mittelstufe-Geschichte/app/index.html
```

**Oder direkt:** Doppelklick auf `app/index.html`

## ✨ Features

### 🎓 Lernen
- **💬 KI-Tutor** - Intelligenter Chat mit simulierten Antworten
- **📝 Operatoren-Training** - AFB I, II, III Übungen
- **🧠 Lernstrategien** - 10+ bewährte Methoden
- **📅 Zeitstrahl** - Interaktive Geschichts-Timeline

### 📚 Üben
- **🎯 Quizzes** - Multiple Choice zu allen Themen
- **📖 Themenübungen** - 320+ Aufgaben (16 Themen à 20 Übungen)
- **🚀 Adaptive Session** - KI-gestützte Lerndiagnostik
- **⏱️ Pomodoro-Timer** - Strukturiertes Lernen

### 📖 Materialien
- **📚 Glossar** - 56 Fachbegriffe (Klasse 8-10)
- **🗂️ Library** - Kategorisierte Lernmaterialien
- **📄 Lernzettel-Upload** - Eigene Notizen hochladen

### 🎮 Gamification
- **🏰 Burg-System** - Fortschritts-Tracking
- **🐄 Kuh-Münzen** - Währungssystem
- **🏆 Achievements** - Erfolge freischalten
- **📊 Rang-System** - Tagelöhner → Legende

### 🎨 Personalisierung
- **👤 Avatar-Editor** - Millionen Kombinationen
- **🌈 Themes** - Grünes Farbschema (lernfördernd)
- **⚙️ Einstellungen** - API-Keys, Präferenzen

## 🔧 Installation (für Entwicklung)

1. **Repository klonen:**
   ```bash
   git clone https://github.com/JohannaRigling/Seminarkurs-Lernapp-Mittelstufe-Geschichte.git
   ```

2. **App öffnen:**
   ```bash
   cd Seminarkurs-Lernapp-Mittelstufe-Geschichte
   # Dann app/index.html im Browser öffnen
   ```

3. **Live-Server (optional):**
   ```bash
   # Mit Python
   cd app
   python3 -m http.server 8000
   # Dann: http://localhost:8000
   ```
