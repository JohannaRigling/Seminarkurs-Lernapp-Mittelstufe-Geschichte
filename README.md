# History Learning App

## Overview
The History Learning App is an interactive educational tool designed for middle school students at a gymnasium. This application aims to enhance students' understanding of historical events, figures, and concepts through engaging lessons and quizzes.

## Features
- **Interactive Lessons**: Students can explore various historical topics through well-structured lessons.
- **Quizzes**: After studying the lessons, students can test their knowledge with quizzes that provide instant feedback.
- **User-Friendly Interface**: The app is designed to be intuitive and easy to navigate, making learning enjoyable.

## Project Structure

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd history-learning-app
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage
- Open `src/index.html` in a web browser to start using the app.
- Navigate through the lessons and take quizzes to test your knowledge.
